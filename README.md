# Hacker News Client

Hacker News Client based on [HackerNews API](https://github.com/HackerNews/API)

## Teck stack
- [Next.js](https://nextjs.org/)
- [Chakra UI](https://chakra-ui.com/)
- [HackerNews API](https://github.com/HackerNews/API)
